# Housing Price Prediction

## Overview
This project predicts housing prices using regression models in Python.

## Dataset
- Source: [Kaggle Housing Prices Dataset](https://www.kaggle.com/)

## Approach
- Data cleaning and preprocessing
- Feature engineering
- Model training with Linear Regression, Decision Tree, and Random Forest
- Evaluation using RMSE and R²

## Results
- Best model: Linear Regression with R² = 0.65
